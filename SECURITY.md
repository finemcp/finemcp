# Security Policy

## Supported versions

We aim to support the latest major version of finemcp and, for a limited time, the previous major version with security fixes.

## Reporting a vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, email the maintainers at: hi@midhundev.asia, 

Please include:
- A description of the issue and potential impact.
- Steps to reproduce, if possible.
- Any relevant environment details.

We will:
- Acknowledge receipt of your report.
- Investigate and determine the impact.
- Work to release a fix and coordinate disclosure.
